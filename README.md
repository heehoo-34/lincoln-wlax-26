<b>Yearly Updates</b> <br>
You will need to change the 'Our Team', 'Schedule', and 'Registration' pages. I recommend that you create a new image folder in order to do so. You can start by creating a new folder in your File Explorer. Name it something like "images[your year]". Once you have put all of the images that you need into that folder, go into the "code" section of the repository, click the "Add File" button on the top right, and select "Upload files" from the dropdown. You can then drag your file into the box and it will upload into the repository. From there, you can start updating the actual html documents.<br>

<i>Team Info</i> <br>
NOTE: Please make sure that you have all of your images in that image folder and the folder in this repository before going through with this section.
<ol>
  <li>Open the document "our-team.html" and click the pencil icon in the top right that says "Edit this file" when you hover over it.</li>
  <li>Scroll to where you see <blockquote>
	&lt;div class="text-captains"&gt; <br>
	&emsp; &lt;img src="images/evelyn.jpeg"&gt; <br>
	&emsp; &lt;p&gt;"Hi! My name is Evelyn Bieniawski and I’m a Senior at Lincoln..."&lt;/p&gt; <br>
	&lt;/div&gt; <br>
</blockquote></li>
  <li>Change the image by replacing "images/evelyn.jpeg" with "[your folder name]/[the file name of your captain's picture]".</li>
  <li>Change the captain's statement by replacing everything between " &lt;p&gt;" and "&lt;/p&gt;" with the new statement that your captain wrote.</li>
  <li>Repeat for the other captains.</li>
  <li>Save the changes by clicking the green button in the top right that says "Commit changes...". The site will automatically update in a few minutes.</li>
</ol>
